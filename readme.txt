Import high-quality responsive email templates, build email, send to your subscribers and track user activity.


== Description ==

https://www.youtube.com/watch?v=IBYmTCyCGf8

This plugin require **[Page Builder by AZEXO](https://wordpress.org/plugins/page-builder-by-azexo/)** and **[Form Builder by AZEXO](https://wordpress.org/plugins/cost-calculator-by-azexo/)**.

= Main Features =

* **Responsive email Drag & Drop composer**
* **Unlimited subscribers** with statistics 
* **Unlimited newsletter** with tracking
* Customizable **subscription widget**, **page** or **custom form**
* SMTP-Ready 
* **Compatible with Postman, WP Mail SMTP, Easy WP SMTP, Easy SMTP Mail, WP Mail Bank**


== Installation ==

1. Upload the `azh_email_marketing` folder to your plugins directory (e.g. `/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the instructions
